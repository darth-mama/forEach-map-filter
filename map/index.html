Map

*Creates a new array

*Iterates through an array

*Runs a callback function fo r each value in the array

*Adds the result of that callback function to the new array

*Returns the new array

*map always returns a new array of the same length

*Must have a return statement, otherwise, when the function is called, the output is 'undefined'

*Common Case Scenario in using maps:

You have a complicated data struture, and you want to extract one or two pieces
